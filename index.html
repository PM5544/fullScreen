fullScreen

A small JavaScript to feature detect and normalize the fullscreen DOM API

Download .zip Download .tar.gz View on GitHub

Usage

  1. Include the fullscreen.js script in your page
  2. either
    • use the default classes and id to auto show/hide content and bind eventListeners depending on the fullScreen support, like in the test page.
      If you use it like this you should also add the styling from the test page.
    • or make use of the API which is returned to populate the fullScreen object
